# Kenya Agricultural Production Analysis

A Monitoring, Evaluation, and Learning (MEL) and Data Analytics project. This analyzes Kenyan county-level agricultural and demographic data to provide insights for sustainability and food security planning.

## Project Overview
- **Objective**: Transform raw data into strategic insights via cleaning, descriptive analysis, visualizations, and policy metrics.
- **Parts**:
  - Part 1: Data cleaning and descriptive stats (e.g., total farming households: 6.35M).
  - Part 2: MEL logic, primary sector classification, and metrics (e.g., Crop Intensity, Agricultural Engagement Rate).
- **Key Outputs**: Cleaned data, visualizations (PNG), summary stats, and a presentation outline.

Data sourced from Kenyan county agricultural datasets (2019). See `docs/presentation_content.md` for full details.
